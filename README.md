# elves-shoemaker
**「小人の靴屋」系・自走型AIエージェント作成キット**

## 命名の由来
グリム童話の「小人の靴屋」みたいに、あなたが寝ている間にAIがコツコツと作業を進めて、朝起きたら「欲しかったプログラム」が出来上がっている……そんな世界を目指して作りました。

## 仕組み：一歩ずつ、確実に。
このシステムは、難しいことを一度にやろうとしません。「今、目の前にある一つのこと」に集中して、階段を一段ずつ登るように進みます。

1. **「今」だけを見る**: 過去の長いやり取りを全部送るのではなく、「今の目標」と「さっきの結果」だけをAIに伝えます。これでAIが混乱するのを防ぎます。
2. **難しいことは分ける**: 大きすぎる問題にぶつかったら、その場で「小さな作業」に細かくちぎって、一つずつ片付けていきます。
3. **ちゃんと確認する**: 一つの作業が終わるたびに、「本当にこれで合ってる？」と自分でチェックします。ダメだったら、自分で考えてやり直します。

## 動きかた（イメージ）
AIは一回につき、一つの道具（ツール）だけを使います。これを繰り返すことで、安全に作業を進めます。

```typescript
while (taskStack.isNotEmpty()) {
  // 1. 次に使う「道具」を選ぶ
  // （「まずはファイルを読もう」「次は書き換えよう」など）
  const effectName = await orchestrator.selectNextEffect();

  // 2. その道具を実際に動かす
  // （ファイルに書き込んだり、ネットで調べたりする）
  await orchestrator.dispatch(effectName);

  // 3. うまくいったか見て、次のループへ
}
```

## 頼りになる道具たち
AIはこんな道具を使って、あなたの代わりに働きます。

- **タスク管理**: 計画を立てる、作業を細かく分ける、ちょっと待機する。
- **ファイル操作**: プログラムを読む、書き換える、一部だけ直す。
- **ネット検索**: **Brave Search** を使って、最新の技術や情報をローカルから調べる。
- **ご主人さまへの連絡**: どうしても困ったときは **Discord** であなたに「助けて！」と泣きついたり、新しい機能をおねだりしたりします。
- **じっくり考える**: いきなりコードを書かずに、まず「どう作るのが正解か」を理論立てて考えます。

## いいところ
- **ローカルで動く**: あなたのPCの中だけで完結するので、大事なコードが外に漏れる心配がありません。
- **お利口なJSON処理**: AIがたまに出す「余計なお喋り」を無視して、命令だけを賢く受け取ります。
- **16GBのビデオカードでOK**: 情報を整理して送るので、家庭用のPCでもサクサク動きます。

## 用意するもの
- **ビデオメモリ(vRAM) 16GBくらい** (RTX 3060 / 4060 Tiなど)
- **LM Studio** や **Ollama** など
- **Node.js**
- **Brave SearchのAPIキー**（ネット検索をさせる場合）
